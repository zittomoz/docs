---
title: "Alias"
description: "Aliases are efficient way for writing commands."
---

To add alias you simply need to use this command

<Tabs>
  <Tab title="Syntax">
    ```
    -alias add <command> -$ <alias>
    ```
  </Tab>
  <Tab title="Example">
    ```
    -alias add ban -$ hammer
    ```
  </Tab>
</Tabs>

### How to see them?

use 

<Tabs>
  <Tab title="Syntax">
    ```
    -alias list
    ```
  </Tab>
  <Tab title="Example">
    ```
    -alias list
    ```
  </Tab>
</Tabs>
